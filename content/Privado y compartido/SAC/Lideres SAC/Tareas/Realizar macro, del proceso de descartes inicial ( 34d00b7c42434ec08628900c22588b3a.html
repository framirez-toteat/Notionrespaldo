1. Descripción

Macro asociada a obtener información de Selftest de las impresoras de red del cliente para validaciones.

2. Alcance

Se realiza proceso para obtener las IP de las impresoras de red del cliente

3. Entrada

Cliente con problemas de impresión/conexión de impresora por el chat de soporte

4. Condiciones

  • Se debe tener en consideración:
    • Este documento se debe obtener solo de las impresoras de red, si lo hace en una impresora USB saldrá información no válida.
    • Si la impresora tiene DHCP activo puede que en el campo IP ADDRESS esté vacío, en ese caso, se debe validar con la MAC del dispositivo realizando un scanner a la red.
    • Verificar si se necesitan las IP de todas las impresoras de red o solo alguna/s en particular.
    • Verificar si tiene IP de fábrica.
    • Verificar la IP del PC servidor, debe coincidir el segmento de red con la de la impresora.

5. Procedimiento

“Estimado, para poder verificar el problema de la impresora de una manera efectiva, le solicitaré que realice el siguiente proceso en la impresora con el problema para así verificar si el problema es de configuración o físico, este consta de obtener un ‘Selftest’ de la impresora, el cual nos muestra toda su configuración interna. Es de vital importancia obtener este documento ya que con él, podemos saber la IP actual de la impresora y validar si el problema está ahí.
Para ello por favor debe realizar los siguientes pasos:

Paso 1: Apagar la impresora.

Paso 2: Apretar el botón Feed (el que hace avanzar el papel en blanco)

Paso 3: Manteniendo el botón Feed presionado, encender la impresora.

Paso 4: No soltar el botón Feed hasta que imprima el Selftest

Paso 5: Apenas salga el Selftest soltar el botón Feed, inmediatamente presionarlo nuevamente por 1 segundo adicional (esto para liberar la acción, botará papel en blanco)

De este documento necesito el apartado IP ADDRESS si no lo consigue, por favor envíeme una foto del papel para ayudarle.

También le dejaré un video con cómo poder realizar esta acción para apoyarle: “

6. Salida

  • Se obtiene el documento necesario para la validación

7. Excepciones del flujo

  • Impresoras sin error.
