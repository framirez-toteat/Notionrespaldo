1. Descripción

Se realiza la creación del protocolo para instalaciones que lleguen al CHAT de Soporte

2. Alcance

Proceso establecido luego de haber pasado por el proceso de Onboarding y encontrarse operativos

3. Entrada

Solicitud de re instalación del PC servidor o impresoras en el CHAT de Soporte

4. Condiciones

  • La persona que reciba la solicitud, debe tener en cuenta:
    • Consultar sí el cliente puede operar o no para determinar la urgencia
    • En caso de tener varios chats, indicarle al cliente que este proceso puede tomar 40-50 minutos
    • En caso de que el cliente no pueda esperar debido a la necesidad de operar, informar inmediatamente a los coordinadores para asignarlo a otra persona o que se autorice estado “Away” para realizarlo.

5. Procedimiento

Validar dependiendo si sólo es instalar impresoras o también se debe instalar el programa en el PC

  • Paso 1: Remote

    Conectarse al escritorio del cliente usando Toteat Remote desde el Tab Support.

  • Paso 2: Instalación impresora
    Preguntar si es una impresora USB o de RED:
    • Win+R+control printers
    • Si hay una impresora instalada, mantener el nombre de la misma.
    • Seleccionar agregar una impresora:
    • Seleccionar “La impresora que quiero no está en la lista”.

    Impresora USB:
    • Seleccionar la opción: “Agregar una impresora local o de red con configuración manual.”
    • Escribir el nombre de la impresora: Ej. “Caja”
    • Verificar la impresora instalada:
    Impresora de Red:
    • Seleccionar la opción: “Agregar una impresora con una dirección IP o un nombre de host”.
    • Verificar IP de la máquina del cliente: Win+R → cmd→ ipconfig
    • Verificar la IP de Impresora:
      • Apagar la impresora
      • Presionar el botón Feed
      • (Sin dejar de presionar el botón feed), prender la impresora, luego soltar el botón de Feed y acto seguido presionar de nuevo.
      • Si se siguen los pasos correctos, saldrá un papel con la configuración “ethernet” de la impresora y apenas salga, presionar el botón Feed.Verificar la IP de Impresora:
    • Verificar la IP de Impresora: Se debe verificar que en qué controlador de red está la impresora conectada, si está en el puerto ethernet (directa o a través de un router), o al controlador WAN, ya que es en este controlador donde realizaremos la configuración de IP en el caso de necesitar una subred, en este caso, la impresora está conectada directamente al puerto Ethernet directo al PC, por lo qué, dejaremos una IP en el segmento de la impresora para tener conexión con ella:
    • Impresora conectada a router

  • Paso 3: Desactivar Firewall

    Para acceder de forma rápida usar los siguientes comandos:

    • Comando wf.msc
    • etsh advfirewall set allprofiles state off

    o bien por:

    • Ingresar al Panel de Control → Sistema y Seguridad→ Firewall de Windows Defender
    • Seleccionar Seguridad Avanzada→ Propiedades de Firewall de Windows Defender e inactivar el Firewall en los tres perfiles (Dominio, privado y público).

  • Paso 4: Verificar el sistema operativo

    Para acceder de forma rápida usar el siguiente comando:

    • En ejecutar ===== >>>>> dxdiag

    o bien:

    • Panel de Control→ Sistema y Seguridad→ Sistema

  • Paso 5: Ambiente
    • Activar el permiso al ambiente del restaurante.
    • Revisar que el ID sea el correcto al momento de ingresar a través del administrador de Toteat, verificar ID de restaurante e ID de local.
  • Paso 6: Instalación TPS
    • Descargar el TPS correspondiente desde el ambiente del cliente: Configuración→ Print Server y API

      Nota: Las opciones de descargas aparecerán una vez que nos demos permisos de admin→ Escribir en la consola: RS.luces.admin=1). Se debe realizar sobre la configuración del Navegador en la consola desarrollador.

    • Ejecutar el TPS descargado para instalar el TPS.
    • Continuar con la instalación con todos la configuración predeterminada:

    Hasta que se complete la instalación

  • Paso 7: Permiso a la carpeta TPS
    • Buscar la carpeta creada, hacer click derecho→ Propiedades → Seguridad.
    • Dar permisos de Control Total a todos los usuarios:
    • Verificar que Totear Print Server haya quedado activo

  • Paso 8: Acceso remoto temporal de Toteat
    • Proceso de verificación de la disponibilidad del PC- Servidor del Cliente para la instalación.
    • Se busca por palabra clave el Local de la Instalación
  • Paso 9: Acceso directo de Toteat
  • Paso 10: Acceso definitivo de remote
    • Seleccionar el Tab Access y luego Build + Completar los datos del acceso y copiar el URL.
    • Copiar el enlace en el navegador del cliente
    • Ejecutar:
    • En caso de que falle, buscar el archivo en descargas y ejecutarlo como administrador.
  • Paso 11: Configurar impresora en ambiente Toteat
    • Configurar impresoras:
  • Paso 12: Imprimir pruebas en cada caja
    • Para realizar este proceso se debe previamente haber abierto turno, abierto caja, y tener una configuración de menú.
    • Finalizado este proceso se deben cerrar los mismos elementos que se abrieron. Y habiendo realizado las anulaciones de los productos facturados.
  • Paso 13: Despedida
    • Mantener SIEMPRE comunicación con el cliente durante la implementación, consultando sobre las pruebas enviadas van saliendo o no en la instalación y en la configuración de reglas.
    • Se deben tener pruebas de las impresiones de todos los puntos.
    • Con todas las pruebas ya realizadas nos despediremos del cliente.

    “Estimado, con estas pruebas damos por terminado el proceso de implementación, recuerde que todo dispositivo ajeno al servidor para utilizar Toteat debe ingresar desde el navegador a res3.toteat.com o res4.toteat.com, para take away, le avisaremos al ejecutivo que la implementación está finalizada, para que puedan continuar con  el proceso, ya sea configuración de ambiente, ajustes de carta y capacitaciones coordinadas, que tenga un buen día y nos alegra mucho poder ser un aliado en su negocio”

    • Se deben anular las pruebas realizadas y ya con esto ir al Slack de OB-SAC y confirmar al ejecutivo etiquetando y dejando en el hilo las fotos de las pruebas de impresión otorgadas por el cliente (siempre solicitar fotos y respaldos de todas las pruebas realizadas)

6. Salida

  • Impresoras instaladas y configuradas, cliente Feliz!
