Cómo detener una impresora térmica que imprime infinitamente desde Toteat (error en Windows)

Paso a paso desde services.msc (Print Spooler / Cola de impresión):

✅ 1. Abre los servicios de Windows

  • Presiona Windows + R en el teclado.
  • Escribe:
    services.msc
  • Presiona Enter.

✅ 2. Detén el servicio de impresión (Print Spooler)

  • En la ventana que se abre, busca el servicio llamado:

    Cola de impresión o Print Spooler.

  • Haz clic derecho sobre él.
  • Selecciona "Detener".

✅ 3. Borra manualmente los documentos atorados

  • Abre el Explorador de archivos de Windows.
  • Escribe en la barra de direcciones (o navega a):

C:\Windows\System32\spool\PRINTERS

  • Si pide permisos de administrador, dale “Sí”.
  • Una vez dentro de esa carpeta, borra todos los archivos que veas.
🔴 ¡Importante! Estos archivos son los documentos atorados que están haciendo que la impresora imprima sin parar.

✅ 4. Reinicia el servicio

  • Vuelve a la ventana de services.msc.
  • Haz clic derecho sobre Cola de impresión / Print Spooler.
  • Elige "Iniciar".

✅ 5. Verifica la impresora

  • Abre el Panel de Control > Dispositivos e impresoras.
  • Haz clic derecho sobre la impresora térmica (ej. POS80, XPrinter, EPSON).
  • Asegúrate de que no esté en “Pausa” ni marcada como “Usar sin conexión”.

✅ 6. Abre nuevamente el POS de Toteat

  • Reinicia el Print server de toteat si es necesario.
  • Ingresa al POS y envía una impresión de prueba

💡 Consejo preventivo:

Para evitar que se repita este problema:

  • Asegúrate de que solo haya una sesión del POS activa.
  • No asignes en TRM (toteat) impresoras que el TPS reconozca pero no estén creadas en Windows
  • Instala siempre el driver recomendado para la impresora térmica.
