TPS
  • No se imprimen las facturas, boletas o precuentas
    • Lo primero que se debe realizar es reiniciar el Toteat Print Server y validar si la impresora está funcional, enviando una página de prueba de Windows. Si el problema continúa:

    Se debe verificar en el menú lateral → Configuraciones → Impresoras, que la regla de impresión de cuentas y precuentas tenga asignada la impresora y que esté seleccionado el modelo correcto. Se recomienda Bixolon SRP 350, aunque podría variar según el modelo. También debemos validar una configuración importante: el tamaño de la fuente debe ser de 11 px / 38 caracteres, ya que este es el tamaño correcto para la impresión de facturas, boletas y precuentas.

  • Me aparece una ventana negra en el pc y no imprime

    Esto es por la ejecución del TPS, se debe ingresar al pc y lo recomendable es usar el comando en la consola del pc;

    TASKKILL /F /IM drone.exe

    netsh advfirewall set allprofiles state off

    cd "C:\ToteatPrintServer\data"

    del /F /S /Q *.lang *.conf *.db *.db-journal

    @echo off

    C:

    cd \ToteatPrintServer

    start Drone.exe

    exit

    Con eso debería solucionarse si no se soluciona hay que reinstalar

  • No se imprimen comandas, precuentas, tickets y facturas (REINICIO TPS)
    • Se debe ingresar al pc y lo primero que se realiza es el reinicio desde la barra de tareas del TPS, dandole clic derecho restart
    • Si no funciona se debe realizar el reinicio del TPS con el siguiente comando en la consola (cmd) abriendola como administrador;

    TASKKILL /F /IM drone.exe
    netsh advfirewall set allprofiles state off
    cd "C:\ToteatPrintServer\data"
    del /F /S /Q *.lang *.conf *.db *.db-journal
    @echo off
    C:
    cd \ToteatPrintServer
    start Drone.exe
    exit

    • Otra posible solución es salir del TPS dandole clic derecho sobre el icono y Exit, luego ir a la carpeta del TPS (C:\ToteatPrintServer) verificar los permisos y por ultimo dentro de la carpeta en la carpeta data eliminar todos los archivos señalados y cambiar el que se llama ops por ops1, salir de la carpeta data y ejecutar el TPS desde el ejecutable como administrador.
    • Por ultimo si no funciona ninguna de las soluciones anteriores se prueba reinstalando el TPS
Inicio de sesión
  • Sesión Caducada o inválida

    Esto ocurre por realizar iniciar múltiples sesiones en toteat para solucionarlo se debe:

    • Cerrar todas las pestañas sesiones abiertas del navegador o acceso directo
    • Borrar datos locales en toteat
    • Abrir solo una sesión por dominio (res1, res2….) y una sola pestaña,
  • No puedo iniciar sesión en toteat (ingreso correo, clave y no inicia)
    • Se debe verificar que el correo con el que está ingresando efectivamente sea parte del staff del local.
    • Validar si le arroja algún mensaje de error como que la contraseña no es correcta. De ser el caso deben recuperarla ya sea si ingresan por google o por toteat login

  • No puedo iniciar sesion, al iniciar con Google no me deja seleccionar mi correo y vuelve a la pantalla de inicio de sesion

    Esto es por la sincronización de el navegador Google Chrome con la cuenta de Google

    Se debe revisar que la Sincronización este desactivada desde los ajustes del navegador

    y tambien debe estar desactivada la siguiente opción para que cuando quieran iniciar sesion los deje escoger la cuenta que deseen

Ajustes generales
  • El turno ha estado abierto mas de “x” horas, se debe cerrar el turno actual y abrir uno nuevo.

    Como indica el mensaje, el turno ha alcanzado el máximo de horas configuradas. Para solucionarlo:

    El perfil dueño idealmente ya que debe modificar la cantidad máxima de horas permitidas con el turno abierto. Esta configuración genera una alerta para evitar que se olvide cerrar el turno y que no se mezclen ventas de diferentes días. Esta modificación se realiza en Menú lateral→Restaurants→ajustes generales→pestaña General

  • Quiero configurar los impuestos para diferentes productos o jerarquias

    Para esto deben dirigirse a Menu Lateral→ Restaurants→ Ajustes Generales, Pestaña de Impuestos, en este lugar pueden agregar los impuestos que deseen y configurarlos para las jerarquias que quieran se puede seguir esta guia

    https://ayuda.toteat.com/toteat/es/articles/2906603-configuracion-interna-impuestos

  • Quiero agregar comentarios a las anulaciones, o a los productos ejemplo; (Sin Sal, Sin vegetales, Etc.)

    Esto se debe realizar en Menu Lateral→ Restaurants→ Ajustes Generales, Pestaña de Comentarios, en esa seccion se pueden agregar comentarios para lo que desea como anulaciones, Descuentos, o directamente para cocina en los productos, ejemplo:

  • Quiero Activar el modulo de inventario En linea, o Stock Control

    Se puede activar el inventario en Linea desde Menu Lateral→ Restaurants→ Ajustes Generales, Pestaña de Inventarios y se activan las siguientes opciones

    Para activar el modulo de stock control debe escribirse directamente al correo de clientes@toteat.com dado a que lo configura el área de customer success

  • Me aparece error al imprimir “fallo token o error al imprimir Boleta”

    Se debe verificar en Menu Lateral→ Restaurants→ Ajustes Generales, Pestaña de Impresoras, si el proveedor es Bsale, debe enviarse con ellos para que le ayuden a revisar que error tienen, si es DTE Toteat debe revisarse si hay algun error con el proveedor de facturación o si tiene configurado algo mal de esta seccion como la impresion de la factura con DTE o la impresion automatica

  • Quiero cambiar la información que aparece en la factura, como Direccion, Razón social, etc.

    Debe ir a Menu lateral→ Restaurants→ Ajustes Generales, Pestaña de Impresoras, y puede cambiar la información en este apartado

  • Quiero cambiar la información de la precuenta, y agregarle un QR

    Debe hacerlo en Menu Lateral-Restaurants-Ajustes Generales, Pestaña de Impresoras, y la ultima seccion puede agregar información a la precuenta, y agregar el URL del QR que deseen

  • Quiero Crear nuevas formas de pago

    Debe ir a Menu Lateral→ Restaurants→ Ajustes Generales, Pestaña de Forma de pago, al darle al boton señalado desplega varias formas de pago personalizadas, o la ultima que aparece es Custom y pueden cambiarle el nombre como deseen

  • Quiero crear una forma de pago que no genere factura

    Menu Lateral→ Restaurants→ Ajustes Generales, Pestaña de Forma de pago, ingresa a la forma de pago con el lapiz y debe dejar la opción señalada marcada, ejemplo:

    esto porque el sistema toma la configuración al reves, es decir si esta marcada o activada NO EMITE comprobante fiscal (Factura o Boleta), Si la opción está desmarcada o desactivada SI EMITE comprobante fiscal (Factura o Boleta), esto solo debe configurarlo el usuario por protocolos de seguridad

  • Quiero configurar permisos a mi personal, agregar o quitar permisos

    Deben hacerlo desde Menu Lateral→ Restaurants→ Ajustes Generales, Pestaña de Seguridad, en este apartado deben revisar cual es el permiso que desean agregar o quitar, se debe tener en cuenta que si agrega el permiso para alguien o para un perfil solo dicho usuario o dicho perfil tendra el permiso, Ejemplo:

    Si los usuarios o perfiles aparecen en rojo quiere decir que a esos usuarios o perfiles no se les permite hacer dichos movimientos, pero a los demás que tengan permisos de hacerlo por su perfil si, ejemplo:

    Y si se pone NADIE en el campo, quiere decir que ningún usuario podra hacer cambios en lo que se configure, solo el perfil Dueño/Creador, ejemplo:

  • Necesito que se imprima ticket para mis pedidos delivery, donde se visualicen los productos y la información del cliente, como nombre, dirección, teléfono…

    Para activar la copia de Despacho, dirígete al menú lateral y selecciona Restaurants → Ajustes Generales. En la pestaña General, desliza hacia abajo hasta encontrar la opción Copia de Despacho. Actívala y luego configura la impresora donde se imprimirá, así como la cantidad de copias necesarias.

    En esta copia despacho no se refleja el precio o total a pagar, solo datos del cliente, previamente seleccionados al configurar y el detalle de los productos o extras.

  • Necesito que se imprima ticket para mis pedidos delivery, donde se visualicen los productos y la información del cliente, como nombre, dirección, teléfono, pero además el costo de los productos y extras y el total a pagar

    Para activar la copia Caja, dirígete al menú lateral y selecciona Restaurants → Ajustes Generales. En la pestaña General, desliza hacia abajo hasta encontrar la opción Copia Caja. Actívala y luego configura la impresora donde se imprimirá, así como la cantidad de copias necesarias.

Turnos
  • No permite abrir turno

    En ocasiones no les permite abrir turno porque están mal configurados, cómo por ejemplo que el día no este seleccionado.

    En ese caso tendrían que ir a Menú Lateral→Restaurante→Config. Turnos y realizar la configuración de cada turno que necesiten, si es más de uno, se recomienda usar esta guía;

    https://ayuda.toteat.com/toteat/es/articles/919514-como-crear-y-modificar-un-turno

  • Cerraron el turno y no les permite abrirlo de nuevo

    Muchas veces cierran el turno creyendo que pueden abrir algún otro turno o cierran el turno por error y en ese caso solo se puede reabrir.

    • Primero que todo debe irse a verificar si tienen más de un turno creado, si solo tienen un turno creado pueden reabrirlo o el usuario Dueño/Creador, también puede reabrirlo Menú Lateral→Operaciones→Reabrir Turno
    • En caso que tengan más de un turno en la configuración deberán preguntarle al cliente que turno van a usar y verificar muy bien el turno que el sistema deja reabrir y los turnos que tienen configurados, los pueden visualizar y diferenciar con el numero del turno, siempre estar seguros de cúal es el turno que el cliente quiere abrir detallando los id de cada turno como se muestra, para que no hayan errores.
  • Mi tienda de uber aparece cerrada y tengo el turno abierto en toteat
    • Verificar que el horario de uber esté configurado como sub-turno del turno principal que tenga abierto
    • El sub-turno de uber debe tener seleccionado el chek en la letra D (Delivery)

Mesas
  • No puedo descarga el archivo CSV de mesas , ni agregar mesas nuevas

    Esto ocurre cuando eliminan un sector estando asignadas mesas en él:

    • Se debe verificar que sectores asignados tienen las mesas en menú lateral→ Restaurants→ Mesas (ejemplo)
    • Comparar que los sectores que tengan asignadas las mesas efectivamente estén creados en menú lateral → Restaurants → Sectores (El sector 70 no existe) lo eliminaron y esto causa el error en las mesas.
    • Se debe crear nuevamente el sector 70 para que reaparezcan las mesas asignadas a ese sector. Después de esto, será posible agregar más mesas y descargar el archivo CSV.

  • Los nombres de mis mesas no se muestran en su totalidad

    Esto sucede porque el nombre de las mesas no debe super el máximo de 5 caracteres incluyendo espacios

  • No me deja agregar más mesas, me aparece un letrero que dice que me debo contactar con el área comercial

    Esto es porque el plan que maneja ya no deja agregar más mesas, para esto debe pagar para poder agregar más, indicarle que debe escribir al correo de clientes@toteat.com solicitando la creación de más mesas en el correo le indicaran tema de costos y demás información

  • Como anular productos de una mesa

    Para eliminar producto de una mesa, para esto en la mesa deberia aparecerte un boton de una basurita algo así

    Al darle clic te saldra un cuadro en el cual debes escribir un comentario y darle a anular

    Cuando el producto este de color rojo le debes dar a confirmar en la parte de abajo

    ya cuando esten anulados todos los productos en la mesa te aparece “Liberar Mesa”

Cajas
  • Al intentar cerrar la caja, no se pueden visualizar las opciones de pago necesarias para completar el cierre

    Verificar el tipo de cierre configurado en la caja, ya que, si está configurado como cierre ciego, solo debe ingresarse el monto en efectivo disponible en el momento del cierre de caja.

  • Necesito reabrir mi caja, pero no encuentro la opción disponible
    • Es necesario verificar que se esté intentando reabrir la caja con el mismo usuario con el que se estaba operando previamente
    • Hay que validar si cuenta con los permisos correspondientes en el apartado de seguridad para poder realizar la reapertura de la caja
  • No logro visualizar mi caja, para poder hacer el cierre de la misma

    Para ello debes ir a menú lateral→Restaurants→Cajas y verificar que la caja esté activa

  • Quiero que los cajeros no vean los montos de cada forma de pago al cerrar su caja

    En ese caso deben dirigirse a Menu Lateral→ Restaurants→ Cajas, ingresar a la caja deseada con el lapiz y modificar la opción señalada como se ve

Personal
  • Cambié a un usuario de perfil pero no tiene acceso a las funciones del nuevo perfil asignado
    • Se debe verificar que efectivamente se haya realizado el cambió del usuario al nuevo perfil
    • De ser positivo el cambio, debe cerrar sesión y borrar datos locales, para que se efectúe el cambio
  • Cree un usuario pero al intentar ingresar no me permite

    Debe revisarse en Menu Lateral→ Restaurants→ Personal que el usuario no este como pendiente, si aparece pendiente es porque no abrio el link que le llego al correo con el que se creo el usuario, ejemplo:

    Si no está pendiente y esta activo deben revisar como estan iniciando sesion y revisar si la sincronización esta afectando en el inicio de sesion del nuevo usuario

    La invitacion dura 48 horas, por lo que si fue enviada en un tiempo mayor, se debe eliminar el usuario y volver a crear, para que le llegue una invitacion nueva

Jerarquía de menú
  • Cree una jerarquía de menú, pero no se ve para comandar en la mesa

    Para que una jerarquía de menú pueda visualizarse en la carta para comandar , es necesario que:

    • Que tenga producto activos, asignados y con precio base
    • Jerarquía de Extras
  • La Jerarquía de menú no se ve en mi carta virtual (Toteat Shop)
    • Verificaremos que, en el menú lateral → Productos → Productos → Jerarquías de menú, la jerarquía tenga marcado el check de 'Visible para clientes”
    • Si la jerarquía de menú sigue sin visualizarse en la Toteat Shop, es necesario validar si los productos de dicha jerarquía tienen algún bloqueo para su visualización. Para ello ingresamos en menú lateral → Productos → Productos, seleccionamos el lápiz de edición.
    • En la pestaña que dice Toteat App/Delivery, observamos que el check de No mostrar en la toteat App, NO esté seleccionado. Si está seleccionado, quitamos el check , luego confirmamos en el botón en la parte de abajo y presionamos guardar productos.
Extras
  • Cómo creo opciones para los productos

    Debes primero que todo crear la jerarquia de extras, Menu Lateral→ Productos→ Jerarquia de extras, en ese apartado se crea la categoria donde van a estar las opciones o extras, se debe llenar el ID nodo, el nombre, y en Min debe elegir el minimo de opciones que podra elegir y en Max el maximo de opciones que podra elegir

    luego de crear dicha jerarquia debe crear los extras que iran a la categoria antes creada en Menu lateral→ Productos→ Extras, debe crear los extras con normalidad igual que como se crean los productos, y el extra debe tener precio base aunque sea un 0

Precios
  • Como modifico los precios de los productos

    Debes dirigirte a Menu Lateral→ Productos→ Productos y debes modificar directamente el campo que dice Precio Base

    Luego de cambiar el precio recuerda grabar cambios.

  • Quiero cambiar el precio del producto y no me permite, no hace nada.

    Cuando no cambie el precio del producto es recomendable que cambie el precio y al nombre del producto le ponga algun espacio o un punto al final, para que se edite el maestro de productos y el sistema tome el punto como un cambio.

Inventario
  • ¿Se puede corregir una transferencia entre bodegas?

    Caso de referencia

    Si es una TRANSFERENCIA ENTRE BODEGAS, solo debe ingresar a Stock Control - Inventarios - Históricos - Ahí deberá buscar la transferencia entre bodegas, ingresar a ella desde el icono del lápiz y reversarla. Al reversarla quedara en estado CREADO y podrá ingresar nuevamente a editarla para posteriormente APROBARLA. Con eso se corregirá la transferencia.

    Si es una TRANSFERENCIA ENTRE LOCALES y aun no recepcionaron la mercadería en el otro local, lo que puede hacer es en el panel de RECEPCION poner las 79 y no las 80, con eso quedara registro en el sistema de que en la transferencia fijaron 80 pero realmente se ingresaron al inventario 79. En caso de que ya hayan confirmado y recepcionado la transferencia, no hay nada que hacer a nivel de esa transferencia, dado que fue despachada y recepcionada.

  • Como se activa y configura un KDS

    Lo recomendable es que nosotros configuremos el KDS dado a que a los clientes no les deja generar el Token de acceso, Menu Lateral→ Configuración→ Kitchen Display System, se activa y se va a la pestaña de Tipos de KDS y se le da a Agregar tipo de KDS

    Luego de haberlo configurado debe indicarsele al cliente que configure el KDS, ingresando con perfil Dueño/Creador, se le puede enviar la siguiente guia para que sea mucho más rapido y entendible para el cliente

    https://ayuda.toteat.com/toteat/es/articles/5436867-como-activar-kds-en-un-equipo

  • Configurar KDS en local Migrado

    Lo recomendable es que nosotros configuremos el KDS dado a que a los clientes no les deja generar el Token de acceso, Menu Lateral→ Configuración→ Kitchen Display System, se activa y se va a la pestaña de Tipos de KDS y se le da a Agregar tipo de KDS.

    Luego de haberlo configurado debe indicarsele al cliente que configure el KDS, ingresando con perfil Dueño/Creador, debe ser con perfil de Dueño/Creador si o si porque es el unico que lo puede configurar y debe hacerlo desde, puede funcionar la misma guia que este en el paso anterior pero cambiando el link por el que se indica

    https://kitchendisplay.toteat.com/

  • Que hacer cuando no deja activar el KDS en un equipo a un usuario
    • Primero se debe verificar que lo este haciendo el usuario Dueño/Creador, si no lo esta haciendo el usuario mencionado no lo va a poder configurar nadie más.
    • Otro punto a revisar es que se elimine cache de la aplicacion en el Menú Lateral aparece la opción, y luego de eso regenerar el Access Token y que intente activarlo de nuevo
    • por ultimo podria usar otro dominio como https://kds1.toteat.com/  o https://kds2.toteat.com/ 
    • Recuerden que el KDS se usa con la sesion cerrada si inician sesion en el KDS se daña la configuración y lo tienen que volver a activar
