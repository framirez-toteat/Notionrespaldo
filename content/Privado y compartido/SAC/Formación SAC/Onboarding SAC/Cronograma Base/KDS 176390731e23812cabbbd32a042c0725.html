  • 🍽️ ¿Cómo habilitar el KDS?

    El KDS (Kitchen Display System) 📺 es una pantalla que muestra en tiempo real el estado de preparación de las órdenes y platos, ayudando a que la cocina sea más ágil y organizada 🚀.

    Para activar el KDS en tu local:

    👤 Una persona con perfil de Dueño o Creador debe enviar un correo a clientes@toteat.com ✉️ o mediante el chat de soporte.

  • 🍽️ ¿Cómo agregar un Tipo de KDS?

    Para utilizar el KDS (Kitchen Display System) necesitas tener al menos un Tipo de KDS agregado.

    Cada Tipo de KDS representa una configuración distinta para la visualización de pedidos en cocina 👨‍🍳👩‍🍳.

    🚀 Pasos para agregar un Tipo de KDS:

    1. Entra al Menú lateral en TRM (Toteat Restaurant Manager).
    1. Dirígete a Configuración → Kitchen Display System.
    1. Selecciona la pestaña Tipos de KDS y presiona el botón ➕ Agregar Tipo de KDS.

    🛠️ Configuración de un Tipo de KDS:

    • Activo: Permite seleccionar si este Tipo de KDS estará disponible para su uso ✅.
    • Nombre: Nombre identificativo del KDS (Ejemplo: Cocina Caliente 🔥 o Pastelería 🍰).
    • Descripción/Ubicación: Define el propósito o ubicación de este KDS 🏠.

    🔄 Estados habilitados en este KDS:

    • Selecciona qué estados estarán disponibles en el KDS.
    • 📌 El último estado seleccionado definirá cuándo un producto desaparecerá de la pantalla.
    • Seleccionar mínimo dos estados.

    🔧 Opciones avanzadas:

    • Permite saltar a estados subsiguientes: Puedes omitir ciertos estados al actualizar pedidos ➡️.
    • Permite volver a estados anteriores: Solo si el producto aún no alcanzó su estado final 🔙.
    • Mostrar productos anulados: Visualiza productos cancelados 🚫.
    • Confirmar recepción de anulación: Permite interacción con productos anulados.

    ⏰ Configuraciones de Tiempos:

    • Tiempo Total mínimo para alerta de atraso: Segundos para cambiar a alerta de retraso ⏳.
    • Tiempo de Advertencia: Segundos para cambiar a advertencia ⚠️.

    🛍️ Configuración de Visualización:

    • Tipo de pedidos a mostrar: Elige qué pedidos se verán.
    • Modo Solo Visualización: Permite solo ver pedidos sin modificar 🔒.
    • Bloquear ajustes de equipo: Impide cambios desde kds.toteat.com.

    ⚙️ Tipo de Configuración:

    • Según Impresoras:

      ➔ Muestra productos según las impresoras configuradas previamente en Configuración → Impresoras 🖨️.

    • Según Reglas Sector/Productos:

      ➔ Permite definir reglas específicas de qué productos o sectores mostrar.

    🔑 Finalizar:

    1. Presiona Grabar 💾.
    1. Luego, presiona Generar Access Token 🔐 y vuelve a grabar.

    ¡Listo! Ya tienes tu Tipo de KDS configurado y preparado para activarlo en los equipos correspondientes 🎉.
