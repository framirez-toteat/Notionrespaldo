1. Descripción

Una vez que se han recopilado los datos necesarios para configurar la integración entre Toteat y la plataforma de PeYA, el siguiente paso es iniciar el proceso de activación. Esto implica configurar la API correspondiente para establecer la conexión entre ambas partes y coordinar el proceso de activación a través del canal de comunicación que mantenemos con PeYA.

2. Alcance

Al activar la integración entre PeYA y Toteat, nuestros clientes podrán disfrutar de las funcionalidades principales que ofrece esta integración. Entre ellas se encuentran la sincronización de menús, la gestión de pedidos y la generación de reportes consolidados.

3. Entrada

Para iniciar el proceso de activación, el cliente debe realizar la solicitud a su agente de Onboarding o de Customer, según el estatus del restaurante en Toteat.

  • En el caso de un cliente nuevo, el agente de Onboarding debe validar durante el proceso si se llevará a cabo la activación de la integración. Es importante destacar que esta información debería haber sido comunicada por el cliente al comercial durante la negociación, para que el servicio quede especificado en Salesforce.
  • Si el cliente ya está en cartera, debe enviar la solicitud al correo clientes@toteat.com, donde recibirá una respuesta con los pasos a seguir.

4. Condiciones

  • La persona que realice y/o solicite este proceso debe tener en consideración:
    • Ser personal autorizado de Toteat para la activación de los procesos de integración y tener conocimientos sólidos sobre el proceso de activación en la integración de PeYA-Toteat.
    • Confirmar con el cliente que ya está creado y activo en PeYA, además de contar con su autorización para iniciar el proceso.
    • Es crucial verificar que el cliente no esté vinculado a otro punto de venta (POS) antes de comenzar el proceso de integración.
    • El cliente debe contar con los datos necesarios para la integración, que incluyen: ID de la tienda, precios por lista para PeYA.
    • Validar con el cliente que los productos tengan imágenes.
    • Validación si es tienda única o esta vinculada.
    • Programar la integración para inicios de semana e informar al cliente que el tiempo de activación es de 1 a 3 días hábiles.

6. Salida

  • La integración con PedidosYa es fundamental para mejorar la experiencia tanto del restaurante como de la gestión de la plataforma de delivery. Proporciona mayor eficiencia en el proceso, además, facilita y centraliza la operación en temas administrativos. Por lo tanto, es de gran importancia conocer el proceso de activación de la integración con PedidosYa para asegurar una eficiente activación y, por ende, una operación exitosa.

7. Excepciones del flujo

  • Toda integración que no corresponda a Delivery.
  • Restaurantes que aún no se encuentren vinculados en la plataforma de PedidosYa.

8. Puntos de control

  • Auditar el proceso de activación de la integración de PedidosYa en todos los países donde aplique.
  • Validar status de la integración y por cual integraciones se realizó.
