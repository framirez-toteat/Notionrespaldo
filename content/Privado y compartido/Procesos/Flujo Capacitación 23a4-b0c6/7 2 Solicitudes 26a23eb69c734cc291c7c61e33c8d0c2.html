1. Descripción

Este proceso permite gestionar de manera eficiente la solicitud de mercadería entre diferentes puntos de venta o hacia un centro de producción. A través de esta herramienta, se facilita la coordinación y el reabastecimiento de insumos o productos entre ubicaciones, asegurando que el inventario se mantenga equilibrado en todos los establecimientos involucrados. El sistema permite solicitar mercadería de manera controlada y rastrear el flujo de productos entre los distintos puntos, optimizando la logística interna de la empresa.

2. Alcance

El proceso cubre la solicitud y transferencia de mercadería entre puntos de venta o hacia centros de producción dentro de la red de la empresa que trabajen con Toteat. El proceso asegura la correcta gestión y coordinación entre los locales, permitiendo que los establecimientos puedan solicitar insumos o productos cuando lo requieran. Además, el sistema rastrea y documenta el flujo de mercancías entre los puntos de origen y destino, garantizando un control eficiente del inventario.

Quedan fuera de este alcance la gestión de pagos, el transporte físico de la mercadería y las solicitudes a proveedores externos.

3. Entrada

Para iniciar este proceso, se requiere la siguiente información:

  • Insumos configurados en los ambientes relacionados.
  • Habilitación y configuración para enviar y recibir.
  • Datos en versiones completados.
  • Configuración previa en el sistema que permita la conexión entre los establecimientos para la solicitud y transferencia de mercadería.

Esta información es fundamental para asegurar que el proceso de solicitud y transferencia de mercadería se ejecute de manera eficiente y controlada.

4. Condiciones

  • La persona que realice y/o solicite este proceso debe tener en consideración:
    • Asesor Toteat para el proceso.
    • Ambientes en versiones en estado mínimo “Periodo de pruebas”.
    • Datos de facturación en versiones rellenados.
    • Insumos y ambientes configurados para el proceso.

5. Procedimiento

  • Paso 1: Solicitud
    • Una vez realizadas las configuraciones anteriores, se debe ir a Stock Control → Transferencias Multilocales.
    • Desde "Solicitudes para Recibir" o "Historial de Solicitudes", se podrá realizar una solicitud.
    • Seleccionar al proveedor/Fabrica.
    • Al seleccionar, se dirigirá a la pestaña de Información, donde solo se podrá ingresar un comentario y seleccionar la fecha de entrega solicitada.
    • Luego, se dirigirá a la pestaña de “Solicitud”, donde indicará la cantidad de los insumos o ingredientes a solicitar.
    • Además, se podrá filtrar por jerarquia de ingredientes, según la configuración del sistema.
    • Nota: Las unidades de medida pueden cambiar según la conversión que tenga configurada el insumo en ambos locales.
  • Paso 2: Carga de solicitud

    Manual:

    • El ingreso de la solicitud se puede hacer de manera manual o manera masiva.
    • La manera manual consiste en ingresar las cantidades en la casilla “Cantidad”.
    • Una vez que las cantidades por insumo a solicitar estén ingresadas, se debe desplazarse hacia abajo y presionar "Grabar".
    • Nota: No se debe presionar “Autorizar” o “Autorizar y Despachar. Primero se debe “Grabar”.

    Masivo:

    • En este paso, se debe descargar el archivo XLS.
    • Se abre ese archivo y se completa la columna “Cantidad”, según la UN definida.
    • Una vez lo requerido tenga cantidad, se copia todo y se presiona el botón Paste.

    • Se pegaran las cantidades en las casillas.
    • Una vez que las cantidades por insumo a solicitar estén ingresadas, se debe desplazar hacia abajo y presionar "Grabar".
    • Nota: Se pegará y mostrara SOLO las casillas que tengan una cantidad ingresada en el excel.
  • Paso 3: Envió de solicitud
    • Luego de haber presionado “Grabar”, el sistema redireccionará a la sección “Solicitudes para Recibir”. Donde se visualizará el movimiento creado.
    • ID: Código único asociado a cada movimiento.
    • Fecha Modificación: Fecha de última modificación del movimiento.
    • Proveedor: Punto al cual se está solicitando la mercadería.
    • Líneas: Cada linea es un insumo independiente(Ejem: Coca Cola + Sprite = 2 Líneas).
    • Estado: Fase en la que se encuentra el movimiento.
    • Fecha Entrega/Recepción: Fecha de cuando se recepcionó o entrego el pedido.
    • Creador: Usuario que generó el movimiento.

    Se debe ingresar al ícono del lápiz y dirigirse a "Solicitud", donde será posible ver lo solicitado e incluso agregar nuevos insumos antes de autorizar la solicitud.

    Además, es posible eliminar la solicitud.

    Para poder enviar esta solicitud se debe presionar “Autorizar” de esa manera, la solicitud será recepcionada por el destino.

    Nota: El sistema no emite ningún tipo de mensaje al realizar solicitudes, si no se revisa, no se sabrá que existe alguna.

    Una vez autorizado, el estado cambiará y se debe esperar el despacho desde el proveedor.

6. Salida

  • Control de solicitudes levantadas por ambiente.
  • Control de materia prima entre locales.

7. Excepciones del flujo

  • Cliente no realiza transferencias entre sucursales.

8. Puntos de control
