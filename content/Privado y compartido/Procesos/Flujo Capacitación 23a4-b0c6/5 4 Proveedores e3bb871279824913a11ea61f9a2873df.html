1. Descripción

En este proceso veremos cómo asociar un proveedor a un insumo específico, con el fin de llevar un mejor control interno. Analizaremos:

  • Proveedor
  • Costo neto negociado
  • Unidad de medida del costo anterior
  • Costo neto de distribución (opcional)
  • Porcentaje de descuento negociado
  • Formato de compra permitido en facturas

2. Alcance

El alcance de este proceso abarca la asociación de proveedores a insumos específicos dentro del sistema de gestión de inventario y compras. Incluye:

  • La identificación y registro de proveedores para cada insumo.
  • La configuración de costos netos negociados y sus unidades de medida correspondientes.
  • El establecimiento de costos netos de distribución cuando sea aplicable.
  • La definición de porcentajes de descuento negociados con los proveedores.
  • La especificación de formatos de compra permitidos en las facturas.

Este proceso es fundamental para optimizar el control interno de los insumos, facilitando una gestión más eficiente de las relaciones con los proveedores y los costos asociados a cada producto.

3. Entrada

Para iniciar este proceso de asociación de proveedores a insumos específicos, se requiere la siguiente información:

  • Lista actualizada de proveedores aprobados por la empresa.
  • Catálogo de insumos utilizado en el sistema de gestión de inventario.
  • Información detallada sobre los costos netos negociados con cada proveedor para cada insumo.
  • Unidades de medida estándar utilizadas para cada insumo.
  • Datos sobre costos netos de distribución, si aplican.
  • Registro de los porcentajes de descuento negociados con cada proveedor.
  • Especificaciones sobre los formatos de compra aceptados en las facturas de los proveedores.

Esta información es crucial para realizar una asociación precisa y eficiente entre proveedores e insumos, lo que permitirá un mejor control interno y una gestión optimizada de las compras y el inventario.

4. Condiciones

  • La persona que realice y/o solicite este proceso debe tener en consideración:
    • Poseer los permisos necesarios en el sistema para modificar los insumos de manera masiva o individual.
    • Listado de ingredientes por proveedor configurado en el sistema.

5. Procedimiento

  • Paso 1: Proveedores
    • Estando en el lápiz del insumo, iremos a “Stocks”
    • Aquí podremos configurar internamente cada proveedor. Es importante que la primera opción siempre sea "Todos".
    • Para poder asociar un proveedor en específico, presionamos “Agregar Proveedor”
    • Bajamos, y podremos rellenar los datos que necesitemos.

    • Proveedor: Selecciona un proveedor de la lista.
    • Costo Neto Negociado: Es opcional, pero te ayudará al ingresar la factura de compra. El monto se completará automáticamente y, si el proveedor cambia el precio, se mostrará una alerta.
    • Unidad de medida del costo anterior: Coloca la unidad de medida correspondiente solo si haz ingresado el precio.
    • Costo Neto de Distribución: Opcional.(Actualmente no disponible).
    • % Descuento Negociado: Ingresa el porcentaje de descuento acordado con el proveedor.
    • Formato de compra permitido en facturas: Es crucial completar este campo para evitar errores de unidades de medida al ingresar facturas.
    • Costo interno para manejo de inventarios: Establece el costo que se utilizará internamente para gestionar el inventario.
    • Nombre Producto en Maestro de Proveedor: Se relaciona el nombre del producto del proveedor con el de nuestro sistema, permitiendo la importación de facturas a través de un archivo XML o la carga DTE de Toteat.
    • Código de Producto en Maestro de Proveedor: Se establece una relación entre el código del sistema y el del proveedor, para una carga de facturas.

    Configuración de carga "Automática" de facturas:

    • Dentro de la misma sección de proveedores, en el insumo, una vez agregado un proveedor, se deberá seleccionar el deseado para luego configurar el "Formato de compra permitido en facturas", el "Nombre de producto en el maestro de proveedor" y el "Código de producto en el maestro de proveedor".

    Nota: Este paso es importante cuando necesitamos cargar facturas desde DTE o el archivo XML.

6. Salida

  • Configuración interna avanzada de los insumos.
  • Insumo asociado a su proveedor.
  • Registro de la configuración realizada para futuras referencias o auditorías.

7. Excepciones del flujo

  • Cliente desea llevar el control en los proveedores, y no en los insumos.
