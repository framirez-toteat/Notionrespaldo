1. Descripción

Las subrecetas o preelaborados son preparaciones declaradas en el local, que sirven para complementar una receta o producir porciones de un insumo.

Las subrecetas a nivel sistema se tratan como "extras", por lo tanto, al momento de crearlas, se debe realizar de esa manera.

2. Alcance

Las Subrecetas se utilizan cuando el cliente utiliza sus insumos e ingredientes para producir complementos para sus platos finales, por ejemplo, el producir su propia Mayonesa Casera, donde utilizan sus ingredientes para llegar a un producto final.

3. Entrada

Este proceso no se solicita, sino que se capacita al cliente al respecto. Para ello, es necesario que participe en las capacitaciones personalizadas o gratuitas, donde se bajará la información y la lógica que utiliza el sistema.

4. Condiciones

  • La persona que realice y/o solicite este proceso debe tener en consideración:
    • Se debe tener activo el módulo de inventario avanzado.
    • Debe haber pasado por las capacitaciones previas para poder llegar a este punto.
    • Debe declarar la producción de estas subrecetas en el sistema.
    • Se deben cumplir los procesos anteriores para poder llegar al actual.
    • Se debe tener previamente configurada la base de datos de ingredientes en el sistema.

5. Procedimiento

  • Paso 1: Creación de la jerarquía de extras “Subrecetas”
    • Ir a Menú lateral -> Productos -> Jerarquía de extras y hacer clic en el lápiz de BA.
    • Allí, crear la jerarquía de extras "Subrecetas" con el ID NODO 999.
    • Nota: Es importante que siempre se creen con el ID 999, ya que es un ID único para las subrecetas.
  • Paso 2: Crear el extra/subreceta
    • Ir a Menú lateral -> Productos -> Extras.
    • Allí, crear las subrecetas como si fueran extras. Ingresar el ID Extra, Nombre Extra, precio base y la jerarquía asociada.
    • Una vez creado el extra, iremos al lapiz y le activaremos el modulo de Stock, con el fin de que ya sea inventariable.
    • Nota: Por buenas prácticas, es importante que los ID y el nombre contengan la palabra SUB, como en la imagen del ejemplo.
  • Paso 3: Creación de la SubReceta
    • Ir a Menú lateral → Stock Control → Ingredientes y Recetas →Recetas.
    • Allí, iremos a Extras Sin Recetas y buscaremos nuestra Sub:

    • Le daremos click al lápiz e ingresaremos los campos que pide el sistema (Cantidad a producir, Unidad de medida Cantidad, Porciones/kg, Usar solo en transacciones de transformación).

    • Luego, comenzaremos a cargar los insumos que se están utilizando para poder llegar a la cantidad producida.

    • Siempre guardar los cambios una vez hayan sido realizados.

6. Salida

Con este proceso, se puede llevar un control de las producciones y los insumos utilizados para poder declararlas.

7. Excepciones del flujo

  • Que el cliente no trabaje con subrecetas o producciones.

8. Puntos de control

  • Revisar en Menú lateral -> Stock Control -> Ingrediente y recetas -> Recetas, allí se debe ir a extras sin recetas y ya deben aparecer las subrecetas creadas.
  • Cuando la SubReceta tenga ingredientes, esta pasará a “Extras Con Receta”.
