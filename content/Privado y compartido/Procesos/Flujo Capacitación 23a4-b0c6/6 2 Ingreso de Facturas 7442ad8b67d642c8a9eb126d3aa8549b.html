1. Descripción

El proceso implica la incorporación de facturas al sistema para asegurar una actualización eficiente del inventario. Durante esta etapa, se registran meticulosamente los datos clave de cada factura, incluyendo el último costo registrado. Además, se mantiene un registro exhaustivo de cada compra en la base de datos del sistema. Este enfoque no solo garantiza que el inventario se mantenga al día con las nuevas adquisiciones, sino que también permite un seguimiento preciso de los costos de los productos, facilitando una gestión más efectiva y transparente de los recursos.

2. Alcance

El alcance de este proceso abarca la recepción y registro de facturas en el sistema con el fin de actualizar el inventario y garantizar un control eficiente de las adquisiciones. Incluye la captura de datos relevantes de cada factura, como el último costo de los productos, y la creación de un registro detallado de las compras realizadas. Este proceso asegura que la información del inventario esté siempre actualizada y disponible para consultas, facilitando la gestión de existencias y la planificación de futuras compras. Quedan fuera de este proceso aspectos como la gestión de pagos o la auditoría financiera de las compras.

3. Entrada

Las entradas necesarias para ejecutar este proceso incluyen:

  1. Recepción de Facturas: Recoger todas las facturas de compras.
  1. Verificación de Datos: Comprobar que la información sea completa y correcta.
  1. Ingreso al Sistema: Registrar datos esenciales como último costo y cantidad.
  1. Actualización del Inventario: Actualizar automáticamente el inventario con las nuevas adquisiciones.
  1. Registro de Compras: Mantener un registro detallado en la base de datos.
  1. Verificación Final: Revisar que toda la información esté correctamente ingresada.

4. Condiciones

  • La persona que realice y/o solicite este proceso debe tener en consideración:
    • Poseer los permisos necesarios en el sistema para cargar facturas.
    • Insumos e ingredientes configurados con su proveedor.
    • Unidades de compra configuradas por insumos.
    • Ingredientes/insumos vinculados por ID y Nombre a su proveedor (Opcional).

5. Procedimiento

  • Manera Paso 1: Ingreso Manual
    • En Registro Contable, presionar el boton “Agregar Mov. Inventario”.
    • Una vez aquí, se mostrará la siguiente vista con los proveedores:
    • Selecciona el proveedor y luego hace clic en “Seleccionar”
    • Se accederá al registro de la factura, donde se solicitarán los siguientes datos:
    • Tipo de documento: Indica la categoría del documento fiscal, como factura, boleta o nota de crédito, y determina su tratamiento contable y tributario.
    • Número de Factura: Es el identificador único que asigna la factura.
    • Fecha de Emisión de la factura: La fecha en que se genera la factura.
    • Fecha de Vencimiento (si no está configurada previamente): Indica la fecha límite para el pago de la factura; si no se establece automáticamente, debe ser definida manualmente.
    • Fecha de Pago: Refleja el día en que se realiza el pago de la factura.
    • Estado de la factura: Describe la situación actual de la factura, como "pendiente", "pagada".
    • Forma de pago: Especifica el método utilizado para realizar el pago, como efectivo, transferencia bancaria o tarjeta de crédito.
    • Fecha Contable/Recepción: En que fecha se registrara el ingreso de insumos en el inventario.
    • Bodega de Recepción: Bodega de destino de la mercadería que se está ingresando.

    Nota: La bodega de recepción, por defecto, aparece como "Según bodega de compra de cada producto", lo que significa que toma la configuración previamente realizada para cada insumo (bodega de compra). Si se selecciona otra bodega, toda la factura se asignará a la bodega indicada.

  • Paso 2: Registrar Factura
    • Se comienza ingresando todos los datos solicitados por el sistema.
    • Nota: Si el impuesto no aparece al ingresar la factura, es porque no ha sido configurado en el insumo.
    • Nota: El servicio de distribución se configura en el proveedor como "Recargo General".
    • Nota: Para aplicar los formatos de compra, estos deben estar previamente configurados (Proceso 5.2).

  • Manera 2: Llamar factura desde DTE
    • Para este proceso, debemos tener previamente (Proceso 5.4).
    • Con el punto anterior completado, se deberá ir a Registro Contable y seleccionar "Agregar Mov. Inventario".
    • Selecciona al proveedor y luego, ingresar el numero de factura.
    • Una vez ingresado el numero de factura, aparecera una lupa.
    • Al darle clic nos aparecera el siguiente mensaje:
    • Al presionar importar, nos rellenara todos los campos segun el documento que encontro.
    • Luego, solo presiona grabar y listo.
    • Nota importante: Es necesario trabajar con DTEToteat para poder utilizar la "Lupa" y acceder a estos documentos.
    • Nota importante 2: No se pueden consultar facturas que estén destinadas a otra razón social.
    • Nota Importante 3: Si o si tiene que tener los insumos asociados al proveedor. Proceso 5.4

  • Manera 3: Archivo XML
    • Para este paso, se necesitará el archivo XML, el cual se puede obtener desde DTE Toteat u otro proveedor.
    • Una vez se tenga el archivo, se deberá ir a Registro Contable.
    • Se agrega un movimiento de inventario y se selecciona el proveedor.
    • Una vez dentro del ingreso, dar clic en XML.
    • Tendremos dos opciones, Copiar el Texto o Cargar el archivo desde el pc.
    • Al pegar el texto se visualiza asi:
    • Al darle procesar, se cargara la factura al sistema.
    • Nota: Al realizar este proceso, se debe tener cuidado, ya que el número de factura no se carga automáticamente junto con el archivo XML.
    • Nota: El archivo XML, se obtiene desde el proveedor DTE de cada cliente.

6. Salida

  • Alimentación del stock por insumo.
  • Registro de compras en el sistema.

7. Excepciones del flujo

  • Cliente no cargara facturas al sistema.

8. Puntos de control

  • Luego de haber grabado, ingresar para revisar lo existente y el inventario final.
