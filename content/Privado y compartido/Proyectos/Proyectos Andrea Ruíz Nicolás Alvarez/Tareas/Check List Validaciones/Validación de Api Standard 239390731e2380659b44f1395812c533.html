Para validar la función de cada una de las URL indicadas, puedes hacerlo mediante Postman https://www.postman.com/

GET y POST son dos métodos HTTP utilizados para enviar y recibir datos. GET se usa principalmente para extraer datos, mientras que POST se usa para enviarlos.
Este método debe seleccionarse en Postman como se muestra en el primer recuadro de la imagen, y luego introducir la consulta deseada.

Algunas consultas requieren especificar rangos de tiempo o datos de órdenes. Al completar estos parámetros correctamente, la consulta mostrará la información correspondiente.
